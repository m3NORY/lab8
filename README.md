# lab8

## Simple HTTP File Server & Qt Client  
Описание проекта  
Проект состоит из двух частей:  
 1. HTTP сервер на C++ с использованием Asio  
 2. Qt клиент для загрузки, скачивания и просмотра файлов  
Сервер хранит файлы в памяти (unordered_map), вычисляет хеш для уникальной идентификации и предоставляет простой HTTP API.  


## API
1. POST /files  
 • Описание: загрузка файла  
 • Тело запроса: содержимое файла (plain text или base64)  
 • Ответ: HTTP 200 OK с hash файла в теле
Пример запроса:
curl -X POST http://localhost:8080/files -d "Hello world"  
Пример ответа:  
HASH:123456789  

2. GET /files
 • Описание: получить список всех файлов  
 • Ответ: HTTP 200 OK с перечнем hash, по одному на строку  
Пример запроса:
curl http://localhost:8080/files
Пример ответа:
123456789
987654321

3. GET /files/{hash}
 • Описание: получить содержимое конкретного файла по hash  
 • Ответ:
 • HTTP 200 OK с содержимым файла  
 • HTTP 404 Not Found, если hash не найден  
Пример запроса:
curl http://localhost:8080/files/123456789  
Пример ответа:
Hello world

## Инструкции по запуску
1. Сервер
 1. Открой проект server.pro в Qt Creator или собери через g++:  
g++ main.cpp -std=c++17 -pthread -I asio/include -o server  
 2. Запусти:
./server
 3. Сервер слушает порт 8080.

2. Qt клиент
 1. Открой client.pro в Qt Creator
 2. Собери и запусти проект
 3. Используй кнопки:
 • Upload — загрузить файл на сервер
 • List — получить список файлов
 • Download — скачать файл по hash
